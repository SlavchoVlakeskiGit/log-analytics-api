# Log Analytics API

A backend service built with FastAPI and MySQL for ingesting, querying, and analyzing operational and security logs.

## Current Status

Project setup in progress. Initial architecture includes:

- FastAPI application structure
- MySQL database integration
- SQLAlchemy models
- Alembic migrations
- REST API versioning

## Planned Features

- Log ingestion endpoints
- Filtering and pagination
- Severity and source-based analytics
- Suspicious activity summaries
- JWT authentication
- Docker support
- Automated tests
